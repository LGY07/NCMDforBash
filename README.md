使用方法(一行命令快速运行)：
```
bash $(wget -O- "https://raw.githubusercontent.com/LGY07/NCMDforBash/main/ncmd.sh")
```

使用方法(下载脚本到本地):
```
wget https://raw.githubusercontent.com/LGY07/NCMDforBash/main/ncmd.sh
sudo chmod +x ./ncmd.sh
./ncmd.sh
```
